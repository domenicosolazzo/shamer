##Deploy Instructions

Install `pip` if you haven't already.

```
$ sudo easy_install pip
```
Clone and deploy your `github-s3-auth` instance.

```
$ git clone git@github.com:localytics/github-s3-auth.git
$ cd github-s3-auth
$ sudo pip install requests termcolor
$ ./deploy.py
```
